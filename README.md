# NewsBOOKI
2022 HSU Computer Engineering capstone design - 디딤돌

# 1. 프로젝트 설명
**프로젝트 배경**
2020년 이후 코로나-19로 인하여 대면수업 대신 실시간 온라인 수업, 동영상 수업으로 대체되었습니다. 대면 수업의 부재로 인해 선후배 간의 교류, 교수님과의 소통이 줄어들고 있습니다. 그로 인해 많은 학생들이 정보를 얻지 못하고 있는 상황입니다. 이런 불편함을 해소하기 위해 교내 공지사항이 올라오면 내가 필요로 하는 공지사항의 경우, 미리 키워드 알림을 설정하여 관련 글이 올라왔을 때 알림을 받아 정보를 놓치지 않게 도와준다.

# 5. 기대효과
**편의성**
1. 하는 키워드를 등록하여 해당 공지사항이 올라올 때마다 알림을 통해 공지사항 업데이트 확인 가능하므로, 공지사항을 놓치는 재학생 수 감소
2. 알림을 받으므로서 학생 입장에서 번거롭게 확인하지 않아도 되며, 언제 올라올지 모를 공지사항을 기다리지 않아도 된다. 학교 생활에 필요한 정보에 접근하고 확인할 수 있으므로 더욱 윤택한 학교생활 기대 가능
